em 단위

  • 부모 요소 폰트의 대문자 M 너비를 1em으로 지정. 1em = 16px
  • 글자 크기 ( em ) = 글자 크기( px ) / 16px
  • em 단위가 쓰여지 곳의 폰트사이즈의 배수
  • 부모 요소 모든 폰트사이즈의 배수 X
  • body:16px X h1:1.5em = h1: 24px
  • body:16px X p:1.5em x span:0.8em = span: 19.2px
  • p.width:400px -> width:20em -> 폰트 사이즈 :: 16px X 1.5em X width:20em ->width:480px

rem 단위

  • rem은 처음부터 기본 크기를 지정하고 그것을 기준으로 글자 크기 지정
  • rem : 문서의 기본 값(16px)의 배수
  • :root { //문서 기본 값 변경 font-size:12px; }
  • 16px X width:20rem = width:320px
