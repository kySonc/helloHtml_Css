width, height 속성

실제 콘텐츠 영역의 크기 지정

기본형 width: <크기> | <백분율> | aute

기본형 height: <크기> | <백분율> | aute

속성 값 설명
<크기> 너비나 높이 값을 px(픽셀)이나 cm(센티미터)같은 단위와 함께 수치로 지정
<백분율> 박스 모델을 포함하는 부모 요소를 기준으로 너비나 높이 값을 백분율(%)로 저정
auto 박스 모델의 너비와 높이 값이 콘텐츠 양에 따라 자동으로 결정
실제 콘텐츠 너비 계산하기
  • 모던 브라우저에서 박스 모델의 전체 너비 = width 값 + 좌우 패딩 + 좌우 테두리
  • 인터넷 익스플로러 6에서 박스 모델의 width 값 = 콘텐츠 너비 + 좌우 패딩 + 좌우 테두리
