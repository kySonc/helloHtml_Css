list-style-position

list-style-position 속성

  • 불릿이나 번호를 들여쓰거나 내어쓸 수 있음
  • 기본형 list-style-position : inside | outside
  • <이미지> = url(이미지 경로)

    • 속성 값

  • inside : 불릿이나 숫자를 안쪽으로 들여 쓴다.
  • outside : 기본 값으로 불릿이나 숫자를 밖으로 내어 쓴다.
  • disc : 채운 원
  • circle : 빈 원
  • square : 사각형
  • none : 불릿 없애기
  1. decimal : 1로 시작하는 십진수
  2. decimal-leading-zero : 앞에 0이 붙는 십진수
  3. lower-roman : 소문자 로마숫자
  4. upper-roman : 대문자 로마 숫자
  5. lower-alpha 또는 lower-latin : 소문자 알파벳
  6. upper-alpha 또는 upper-latin : 대문자 알파벳
  7. armenian : 아르메니아 숫자
  8. georgian : 조지 왕조시대의 숫자

    list-style 속성 한꺼번에 표기

  1. ul{ list-style-type:lower-alpha; list-style-position:inside; }
  2. list-style-type, list-style-position, list-style-image 속성을 한꺼번에 표기
  3. ul{ list-style:lower-alpha, inside; }
