list-style-type

list-style-type 속성

  • 순서 없는 목록의 불릿이나 순서 목록의 숫자를 바꾸는 속성
  • 기본형 list-style-type : none | <순서 없는 목록의 불릿> | <순서 목록의 불릿>
  1. disc : 채운 원
  2. circle : 빈 원
  3. square : 사각형
  4. none : 불릿 없애기
  1. decimal : 1로 시작하는 십진수
  2. decimal-leading-zero : 앞에 0이 붙는 십진수
  3. lower-roman : 소문자 로마숫자
  4. upper-roman : 대문자 로마 숫자
  5. lower-alpha 또는 lower-latin : 소문자 알파벳
  6. upper-alpha 또는 upper-latin : 대문자 알파벳
  7. armenian : 아르메니아 숫자
  8. georgian : 조지 왕조시대의 숫자
