list-style-image

list-style-image 속성

  • 순서 없는 목록의 불릿을 이미지로 바꾸는 속성
  • 기본형 list-style-image : <이미지> | none
  • <이미지> = url(이미지 경로)

    • 속성 값

  • none : 이미지를 사용하지 않고 list-style-type 속성에서 지정한 형태로 표시
  • <이미지> : url("images.jpg")처럼 url( ) 키워드 안에 이미지 파일 경로를 지정합니다
  • disc : 채운 원
  • circle : 빈 원
  • square : 사각형
  • none : 불릿 없애기
  1. decimal : 1로 시작하는 십진수
  2. decimal-leading-zero : 앞에 0이 붙는 십진수
  3. lower-roman : 소문자 로마숫자
  4. upper-roman : 대문자 로마 숫자
  5. lower-alpha 또는 lower-latin : 소문자 알파벳
  6. upper-alpha 또는 upper-latin : 대문자 알파벳
  7. armenian : 아르메니아 숫자
  8. georgian : 조지 왕조시대의 숫자
