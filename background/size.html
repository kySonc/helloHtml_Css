background-size 속성

배경 이미지 크기 조절

기본형 background-size: auto | contain | cover | <크기 값> | <백분율>
  • auto : 원래 배경 이미지 크리만큼 표시
  • contian : 요소 안에 배경 이미지가 다 들어오도록 확대/축소
  • cover : 배경 이미지로 요소를 모두 덮도록 확대/축소
  • <크기 값> : 너비 값과 높이 값을 지정. 너비 값만 지정할 경우, 원래 배경 이미지 크기를 기준으로 축소/확대 비율을 자동으로 계산해 높이 값을 지정
  • <백분 율> : 원래 배경 이미지 크기를 기준으로 확대하거나 축소
