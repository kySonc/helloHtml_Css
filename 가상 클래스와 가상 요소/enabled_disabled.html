가상 클래스와 가상 요소

UI 요소 상태에 따른 가상 클래스

:enabled / :disabled

요소를 사용할 수 있을 때와 없을 때의 스타일 지정

표기 설명
:enabled / :disabled 해당 요소가 사용 가능한 상태일 때의 스타일을 지정하려면 :enabled 클래스를 사용하고 해당 요소가 사용 불가능한 상태일 때의 스타일을 지정하려면 :disabled 클래스를 사용. 예를 들어 여러 줄의 텍스트를 입력할 수 있는 텍스트 영역 필드(textarea 필드)를 게시판 입력 화면으로 사용할 때는 enabled 상태이지만 회원 약관 등을 보여 줄 때는 사용자가 내용을 보기만 해야 하므로 disabled 상태로 만들어야 한다
