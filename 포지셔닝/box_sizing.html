box-sizing 속성

박스 모델의 너비 값 기준 지정

기본형 box-sizing : content-box | border-box

속성 값 설명
content-box width 속성 값을 콘텐츠 영역 너비 값으로 사용. 기본 값
border-box width 속성 값을 콘텐츠 영역에 테두리까지 포함한 박스 모델 전체 너비 값으로 사용
