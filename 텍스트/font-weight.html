font-weight

font-weight 속성

  • font-weight: normal | bold | bolder | Lighter | 100 | 200 | 300 | 400 | 500 | 600 | 700 | 800 | 900

    • 속성 값

  • nomal : 일반적인 형태로 기본 값
  • bold | lighter | bolder : 굵게 | 원래 굵기보다 더 가늘게 | 원래 굵기보다 더 굵게 나타냄
  • 100~900사이의 수치 : 400은 normal, 700은 bold에 해당하며 숫자 값을 조절해 좀 더 세밀하게 글꼴의 두께를 조절
  • 백분율 : 부모 요소의 글자 크기르 기준으로 해당되는 %를 계산해 표시. (%와 함께 표기)

font-variant 속성

  • 대문자를 소문자 크기에 맞추어 작게 표시
  • font-variant: normal | small-caps

    • 속성 값

  • normal: 일반적인 형태로 표시
  • small-caps: 작은 대문자로 표시
