font-size

font-size 속성

  • 글자 크기를 조절하는 속성
  • 글자 크기를 조절하는 속성
  • 사용할 수 있는 값: 절대크기, 상대크기, 숫자, 백분율
  • 기본 값은 상대 크기인 medium
  • font-size 속성은 상속된다

속성 값

  • 절대 크기 : 브라우저에서 지정한 글자 크기. 사용할 수 있는 값은 xx-small | small | medium | large | x-large | xx-large
  • 상대 크기 : 부모 요소의 글자 크기(font-size_를 기준으로 더 크게 표시하거나 더 작게 표시. 사용할 수 있는 값은 larger | smaller
  • 크기 : 브라우저와 상관없이 글자 크기를 직접 지정
  • 백분율 : 부모 요소의 글자 크기르 기준으로 해당되는 %를 계산해 표시. (%와 함께 표기)

<크기> 값에서 사용하는 단위

  • em : 해당 글꼴의 대문자 M의 너비를 기준으로 크기를 조절
  • ex : x-height(엑스 하이트). 해당 글꼴의 소문자 x의 높이를 기준으로 크기를 조절
  • px : 픽셀. 모니터에 따라 상대적 크기
  • pt : 포인트.일반 문서에서 많이 사용하는 단위

px 단위

  • px 단위를 사용하면 폰트 크기가 고정됨
  • 모바일 기기로 볼 때도 같은 크기로 화면에 표시되기 때문에 작은 화면 안에 작은 글씨로 표시됨

em 단위

  • 사용하는 글꼴의 대문자 M을 기준으로 한다
  • 대문자 M의 너비를 1em으로 놓고 상대적 값을 계산해 다른 요소들의 글자 크기를 조절
