text-transform

text-transform 속성

  • 영문 텍스트의 대문자나 소문자를 바꾸는 속성
  • 기본형 text-transform : none | capitalize | uppercase | lowercase | full-width;

    • 속성 값

  • none : 변환하지 않습니다
  • capitalize : 시작하는 첫 번째 글자를 대문자로 변환한다
  • uppercase : 모든 글자를 대문자로 변환한다
  • lowercase : 모든 글자르 소문자로 변환한다
  • full-width : 가능한 모든 문자를 전각 문자로 변환한다
